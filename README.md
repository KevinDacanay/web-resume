# Web Resume

## Author
Kevin Dacanay

## Description
Web Resume is a personal website that showcases my professional resume, skills, education, and experience in a clean, responsive web format. In addition to displaying my resume content directly on the site, the application includes a button that allows visitors to view the PDF version of my resume and download it for offline use.

## Credits
All work for this project is original. It was built using HTML and CSS following responsive design principles and modern web development practices.

## Instructions for Compiling and Executing

### Running Locally
This application consists of static HTML, CSS, and JavaScript files. No compilation is required to run the project locally.

1. Clone or download the repository to your local machine.
2. Navigate to the project directory.
3. Open the `index.html` file in any modern web browser (Chrome, Firefox, Edge, Safari).

### Deployment
This project is configured for continuous deployment using Azure Static Web Apps.

1. Changes pushed to the main branch of the GitHub repository automatically trigger a GitHub Actions workflow.
2. The workflow deploys the latest version of the site to the Azure Static Web App instance.

Visit the site @ [https://blue-rock-07640761e.6.azurestaticapps.net/](https://blue-rock-07640761e.6.azurestaticapps.net/)
